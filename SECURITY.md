# Security Policy

## Supported Versions

Security updates apply to the latest release on the `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security reports.

Preferred reporting method:

1. GitHub Security Advisories (if enabled on the repository)
2. Email the maintainers at **jiangkai3108@gmail.com**

We will acknowledge receipt as soon as possible and work on a fix.
